Pytorch based simulators
------------------------

Objecitves
~~~~~~~~~~
This class of augmented simulators are based on Pytorch library. We have provided one baseline implementation for
the simulators based on Pytorch which is a "Fully Connected" architecture.
Users should feel free to use either tensorflow or pytorch libraries.
