Tensorflow based simulators
---------------------------

Objecitves
~~~~~~~~~~
This class of augmented simulators are based on Tensorflow library. We have provided two baseline implementation for
the simulators based on Tensorflow which are : "Fully Connected" and "LeapNet" architectures.
Users should feel free to use either tensorflow or pytorch libraries.
